# RIGN Brand Identity

File: rignbranding.md

This file defines the complete visual identity, user experience, design
philosophy and emotional direction of the RIGN website.

Every page, component and interaction must reference this document
before being created.

The goal is not to build another vehicle wrapping website.

The goal is to build a luxury automotive brand that happens to offer
exterior trim styling.

Chrome delete is one of our services, but our customers are buying a
cleaner, more refined version of their vehicle---not vinyl.

Everything should feel OEM+.

Never aftermarket. Never loud. Never modified for attention. Always
engineered. Always intentional. Always premium.

Customers should feel like they're interacting with Porsche Design,
Apple or Rolls-Royce rather than a local wrapping company.

------------------------------------------------------------------------

# Emotional Goal

When somebody lands on the website they should immediately feel:

-   Silent
-   Confident
-   Premium
-   Modern
-   Engineered
-   Beautiful

The feeling should resemble:

-   Opening the door of a brand new Range Rover.
-   Sitting inside a Rolls-Royce Phantom.
-   Driving through heavy mist at night.
-   Walking through an Audi showroom.
-   Holding a premium aluminium product.

Nothing should shout.

Everything should whisper quality.

------------------------------------------------------------------------

# Design Philosophy

Less. Then less again.

Remove everything that isn't absolutely necessary.

White space is a design element.

Large imagery is a design element.

Typography is a design element.

Motion is a design element.

Every pixel must have a purpose.

------------------------------------------------------------------------

# Visual Style

Everything should feel designed by an automotive manufacturer.

Not a marketing agency.

Not a typical web designer.

Not a vehicle graphics company.

Use:

-   Large photography
-   Large typography
-   Minimal copy
-   Massive breathing space
-   Strong hierarchy
-   Editorial layouts

------------------------------------------------------------------------

# Brand Assets

## Primary Logo

RIGN — angular wordmark, brushed-metal/silver on black

## Tagline

REFINED AUTOMOTIVE — small, letter-spaced, understated

Use whichever version best suits the available space.

Never stretch.

Never distort.

Never add effects.

------------------------------------------------------------------------

# Logo Design Language

-   Sharp, angular geometry
-   Constant stroke width
-   Plenty of spacing
-   Monochrome — black & white / brushed silver only
-   No gradients faking the metallic finish — use the logo as a proper
    image asset
-   No bevels
-   No carbon fibre textures
-   No flames
-   No racing graphics
-   OEM+ styling only

------------------------------------------------------------------------

# Photography

Use aspirational but believable vehicles.

Examples:

-   Audi RS3
-   Audi RS6
-   Golf R
-   BMW M340i
-   BMW M3
-   Mercedes C63
-   Mercedes G-Class
-   Range Rover Sport
-   Range Rover Vogue
-   Defender
-   Porsche Macan
-   Tesla Model Y
-   Tesla Model 3
-   Volvo XC90
-   Cupra Formentor
-   Polestar

Avoid:

-   Ferrari
-   Lamborghini
-   McLaren
-   Bugatti

Photography style:

-   Dark
-   Mist
-   Rain
-   Studio lighting
-   Low reflections
-   Monochrome
-   Close-up details
-   Trim
-   Mirrors
-   Badges
-   Window surrounds
-   Door shuts

Luxury over speed.

------------------------------------------------------------------------

# Typography

Typography carries the design.

-   Large headings
-   Small paragraphs
-   Minimal wording
-   Strong hierarchy
-   Elegant
-   Confident

------------------------------------------------------------------------

# Layout

-   Mobile first
-   Full-width imagery
-   Editorial layouts
-   Large spacing
-   Minimal UI
-   Never overcrowded

------------------------------------------------------------------------

# Motion

Animations should feel expensive.

Use:

-   Fade
-   Blur
-   Parallax
-   Gentle scale
-   Smooth transitions

Avoid:

-   Bounce
-   Flashy effects
-   Spin animations

------------------------------------------------------------------------

# Calls To Action

Primary CTA:

**Get Your Quote**

Always visible in the first viewport.

Maximum two interactions before the quote process starts.

------------------------------------------------------------------------

# Forms

Keep forms effortless.

Only ask for information when necessary.

Prioritise photo uploads.

The experience should feel like messaging a specialist rather than
filling in paperwork.

------------------------------------------------------------------------

# Tone of Voice

Never oversell.

Never use clichés.

Avoid:

-   Industry leading
-   Best prices
-   100% satisfaction

Write like an automotive manufacturer.

Minimal.

Confident.

Premium.
