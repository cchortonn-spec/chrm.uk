## Five highest-impact fixes

   currently absent from the live sitemap, so they cannot earn impressions or
   improve live topical depth.
2. **Reduce homepage mobile LCP.** The lab score fell from 88 to 84 and LCP
   rose from 3.8s to 4.6s, while desktop remained strong.
3. **Replace the short shared town-page structure.** The five current town hubs
   remain the site's largest measured duplicate-content risk.
4. **Deepen visible local answers before adding more schema.** Town FAQs and
   service decisions should be supported by genuinely useful on-page copy, not
   schema-only expansion.
5. **Make sitemap freshness honest and review image semantics.** Replace
   build-time `lastModified` values with route-level content dates when the
   content model supports them, and confirm homepage empty alt text is reserved
   for decorative images.
